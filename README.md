# Sorting Performance Analyzer (SPA)

## Overview
This project is part of Unit–3 (Sorting Algorithms) assignment.  
It implements and compares three sorting algorithms using Python:
- Insertion Sort
- Merge Sort
- Quick Sort

The program analyzes their performance on different types of datasets.

## Features
- Implementation of sorting algorithms from scratch
- Dataset generation (Random, Sorted, Reverse Sorted)
- Performance measurement using time function
- Comparison of execution time for different input sizes

## Files Included
- `spa_sorting.py` → Main Python code
- `report.pdf` → Explanation and analysis
- `output.txt` → Program output (timing results)

## How to Run
Run the following command in terminal:
python spa_sorting.py
